# Naming convention

* Status: proposed
* Date: 2023-06-17

## Decision Drivers

* Many resources inside Azure cannot be renamed later

## Considered Options

* Standard naming convention {resource-abbrev}-{name}-{number}-{region}

## Decision Outcome

Chosen option: "", because comes out best.
